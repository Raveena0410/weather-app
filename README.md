 Weather App

This is a simple weather app I built using JavaScript and the OpenWeather API.
You can enter any city name and it will show the current weather details like temperature, humidity, and condition.

##  What it does

* Search weather by city
* Shows temperature in °C
* Displays humidity
* Shows weather condition (like haze, rain, etc.)
* Uses emojis based on weather
* Handles errors if the city is not found

##  Tech used

* HTML
* CSS
* JavaScript
* OpenWeather API
